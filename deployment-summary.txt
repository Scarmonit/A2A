USAGE INSTRUCTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. GitHub Copilot Integration:
   Add to VS Code settings (.vscode/settings.json):
   
   {
     "github.copilot.advanced": {
       "mcpServers": {
         "a2a-agent-server": {
           "command": "node",
           "args": ["/home/user/A2A/dist/index.js"]
         }
       }
     }
   }

2. Claude Desktop Integration:
   Add to Claude Desktop config:
   
   {
     "mcpServers": {
       "a2a": {
         "command": "node",
         "args": ["/home/user/A2A/dist/index.js"]
       }
     }
   }

3. Direct WebSocket Connection:
   Connect to ws://127.0.0.1:8787 for streaming
